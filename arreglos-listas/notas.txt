La lógica de la aplicación se programa en JavaScript, no en HTMl ni en CSS.

El HTML sirve como intermediario entre el usuario y la aplicación construida en JavaScript.

Un programa se divide en cuatro partes: 
1- Conexiones y estructura lógica. 
2- Entrada de datos.
3- Procesamiento lógico.
4- Salida de datos o resultados.

JavaScript soporta la programación funcional, la programación modular y la programación orientada a objetos. 

En JavaScript se pueden utilizar funciones nativas y funciones definidas por el o la programadora. 

Una función nativa es aquella que viene integrada con el lenguaje de programación y solo necesita ser invocada más no declarada.

Una función definida por el o la programadora es aquella que si necesita ser declarada, definida y programada para luego ser invocada.
